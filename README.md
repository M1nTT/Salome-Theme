#### 《莎乐美》
—— 叶明新


> 莎乐美是希律王的继女
> 
> 长得非常漂亮
> 
> 会跳带有色情意味的
>
> 七层面纱之舞
> 
> 据史料记载
> 
> 她对事物的认知趋于偏激
> 
> 如果你问她什么是爱
> 
> 她只会回答一个字：死
> 
> 施洗者约翰
> 
> 王尔德、海涅
> 
> 理查德·斯特劳斯
> 
> 还有其他人
> 
> 都在不同的时候
> 
> 探讨过这个话题
